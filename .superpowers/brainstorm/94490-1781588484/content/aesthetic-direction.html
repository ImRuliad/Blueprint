A. Terminal Noir

Deep black backgrounds, monospace everywhere, minimal chrome. Feels like a hacker tool that happens to have a GUI. Think: Warp terminal meets TablePlus dark mode. Color-coded connection status dots as the only accent.

Light, airy, stone-warm neutrals. Rounded corners, soft shadows, generous whitespace. Feels like a premium macOS native app — Postico 2 energy but as a web app. Tailwind's stone palette as the foundation. Calm and confident.

C. Midnight Blue

Dark slate backgrounds with blue accent system. Professional and dense — information-rich without feeling cluttered. Feels like a modern IDE (VS Code dark) crossed with a financial terminal. Badge system for engine types and roles. The "serious tool for serious work" energy.

D. Zinc Minimal

shadcn's default zinc palette taken to its purest form. Barely-there borders, square status dots, pill-shaped tabs. No decorative elements — every pixel earns its place. The default shadcn aesthetic but elevated with tight spacing and restrained color. Feels native to the shadcn-svelte stack we're using.
